## Features
1. Navigation Commands
- pwd          # show current location
- ls           # list files and folders
- cd folder    # go into a folder
- cd ..        # go back one folder
- cd ~         # go to home directory
2. File Commands
  # Create
touch file.html          # Mac/Linux
New-Item file.html       # Windows

# View
cat file.html            # print file contents
head -10 file.html       # first 10 lines
tail -10 file.html       # last 10 lines

# Copy
cp file.html backup.html          # Mac/Linux
Copy-Item file.html backup.html   # Windows

# Move/Rename
mv file.html docs/       # move to folder
mv old.html new.html     # rename

# Delete
rm file.html             # Mac/Linux
Remove-Item file.html    # Windows
